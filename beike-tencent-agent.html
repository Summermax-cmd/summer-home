三 · 用户授权机制（核心）

数据不流动，意图才流动

这是腾讯最关心的部分。双方各守边界，只交换结构化的任务上下文，而非原始数据。

🔒

User Memory 不离开腾讯

用户的长期记忆与原始画像始终留在腾讯侧。

🏠

房产数据不离开贝壳

房源、小区、经纪人等核心资产始终留在贝壳侧。

🔁

只交换任务上下文

双方传递结构化意图与结果，而非可还原的个人数据。
